This project focuses on analyzing and predicting Bitcoin prices using Machine Learning and Data Analysis techniques in Python.

---

# 📌 Project Overview

The main objective of this project is to study Bitcoin market trends and predict future closing prices using historical cryptocurrency data.

The project includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Monthly Return Analysis
- Volatility Analysis
- Machine Learning Prediction

The prediction models are trained using historical Bitcoin market data.

